# 기술 스택

## QA / Testing

| 기술 | 경험 |
|------|------|
| Selenium | 웹 UI 자동화 테스트 설계 및 실행 |
| JUnit | Java 단위 테스트 작성 |
| Postman | API 기능/회귀 테스트, Collection 관리 |
| TestRail | 테스트 케이스 관리 및 실행 결과 추적 |
| Playwright | E2E 테스트 자동화 |
| 수동 테스트 | 탐색적 테스트, 회귀 테스트, 크로스 브라우저 테스트 |

## Backend

| 기술 | 경험 |
|------|------|
| Java | 주력 언어. Spring 기반 백엔드 개발 및 테스트 코드 작성 |
| Spring Boot | REST API 설계, 서비스 레이어 구현, JPA 연동 |
| Node.js | Express/Next.js 기반 서버 사이드 개발 |
| Python | 자동화 스크립트, 데이터 처리, 테스트 도구 작성 |
| PostgreSQL | Neon 서버리스 DB 운영, Prisma ORM 활용 |

## Frontend

| 기술 | 경험 |
|------|------|
| TypeScript | 타입 안전한 프론트엔드/백엔드 개발 |
| React | 컴포넌트 기반 UI 개발, 상태 관리 |
| Next.js | App Router, 서버 컴포넌트, API Routes |
| Tailwind CSS | 유틸리티 기반 스타일링, 다크 테마 구현 |

## DevOps / Infra

| 기술 | 경험 |
|------|------|
| Docker | 컨테이너 기반 개발 환경 구성 |
| CI/CD | GitHub Actions 기반 빌드/배포 자동화 |
| Vercel | Next.js 프로젝트 배포 및 운영 |
| Git | 브랜치 전략, PR 기반 코드 리뷰 |

## Tools / Collaboration

| 기술 | 경험 |
|------|------|
| Jira | 이슈 트래킹, 스프린트 관리, QA 워크플로우 |

| Slack | 팀 커뮤니케이션, QA 알림 자동화 |
| Figma | 디자인 리뷰, QA 관점 UI 검수 |
